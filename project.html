Create a SVG logo

This project allows the user to create a SVG logo. SVG stands for scalable vector graphic and it is a web-friendly vector file format that stores images via mathematical formulas based on points and lines on a grid.

Trivia

This trivia game is a group project that uses a modal and 2 APIs: trivia API and random dog photo generator API. The modal allows for the user to choose their trivia category, level of difficulty, and which dog breed they would like to see. When the user answers the trivia prompt, a random dog photo pops onto the screen!

Weather Dashboard

This weather dashboard allows the user to pick any city and see the full week forcast. I used a weather server-side API.
